### Quick Start

| Project | Command |
|---------|--------|
| Stock analysis | `cd agentic-ai && pip install -r requirements.txt && python multi-agent-for-stock-analysis.py` |
| MCP server | `cd mcp/emp-leave-balance && uv init uv-venv && cd uv-venv && uv add "mcp[cli]>=1.6.0" && uv run mcp install main.py` |
| Docker | `cd mydocker/docker-step-by-step/01-Docker-Introduction` |
| Feature flags | `cd feature-flag && pip install -r requirements.txt && python setup_simple_ssm.py && python create_lambda_function.py` |
| Dashboard | `cd panel-demo && pip install -r requirements-dashboard.txt && panel serve sales_dashboard_server_optimized.py --show` |
